# Obsidian Roam Theme

A CSS snippet for Obsidian that recreates the Roam Research aesthetic, designed to work with the [Minimal theme](https://github.com/kepano/obsidian-minimal).

## Features

- **Dark theme** with Roam-style colors
- **Gold headers and links** (#ffc76a)
- **Light blue body text** (#98daf2)
- **Dark navy backgrounds** (#1D1F27, #000911)
- **Operator Mono font** (with fallbacks)
- **Italic internal links** like Roam
- **Styled tags** with subtle background
- **Wider content width** (900px)
- **Mobile padding** to prevent edge-flush text

## Installation

1. Install the [Minimal theme](https://github.com/kepano/obsidian-minimal) in Obsidian
2. Copy `roam-theme.css` to your vault's `.obsidian/snippets/` folder
3. Enable the snippet in Settings → Appearance → CSS Snippets

## Colors

| Element | Color |
|---------|-------|
| Background (primary) | `#1D1F27` |
| Background (secondary) | `#000911` |
| Body text | `#98daf2` |
| Headers & links | `#ffc76a` |
| Tags | `#CB8A1E` |
| Muted text | `#8d9fb9` |

## Requirements

- Obsidian
- Minimal theme (recommended)
- Operator Mono font (optional, falls back to system monospace)
